# Clinic Management System

A React-based clinic management system built with Redux for state management.

## Features

- Patient management
- Appointment scheduling
- Modern React 19 with Redux

## Getting Started

### Prerequisites

- Node.js (v14 or higher)
- npm or yarn

### Installation

```bash
npm install
```

### Running the Application

```bash
npm start
```

Runs the app in development mode at [http://localhost:3000](http://localhost:3000).

### Building for Production

```bash
npm run build
```

Builds the app for production to the `build` folder.

### Running Tests

```bash
npm test
```

## Technologies Used

- React 19
- Redux
- React Redux
- React Scripts
